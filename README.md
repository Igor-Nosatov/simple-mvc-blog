# simple-MVC-blog
Small blog written in pure php.

## Installation
#### Database
- Create a new database then import database.sql  
- Setup the database connection in App/Config/database.php
- Default user is admin / -HAzSdr{gC3LhK?A

#### Development
```
$ npm install
$ npm run build
```
